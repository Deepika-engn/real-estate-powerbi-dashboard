# Real Estate Sales & Market Insights Dashboard

## project Overview
This project is a dynamic Power BI dashboard created to analyze real estate sales, pricing trends, customer behavior, and market demand.

## Tools Used
- Power BI
- Excel
- Power Query

## Features
- Dynamic dashboard
- Interactive slicers
- Revenue analysis
- Price trend analysis
- Demand vs Supply comparison
- Property type distribution

## Key Insights
- Brigade Group generated highest revenue
- Apartments dominate the market
- Indiranagar has highest property prices

## Dynamic Functionality
The dashboard updates automatically when the dataset is refreshed.
